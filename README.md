# 🏡 WanderLust – Full Stack Rental Platform

A full-stack Airbnb-style web application where users can explore, create, and manage property listings.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup / Logout)
- 🏠 Create, Edit, Delete Listings
- 🖼️ Image Upload with Cloudinary
- 🗺️ Map Integration using Mapbox
- 💬 Flash messages for user feedback
- 📦 Session management using MongoDB
- 🎨 Responsive UI with Bootstrap

---

## 🛠️ Tech Stack

### Frontend:
- HTML, CSS, Bootstrap, EJS

### Backend:
- Node.js, Express.js

### Database:
- MongoDB, Mongoose

### Other Tools:
- Cloudinary (image storage)
- Mapbox (maps)
- Passport.js (authentication)
- Connect-Mongo (session store)
